Story contract:
- Add exactly two active command definitions/effects:
  - Marshal Minor Command `rally_order` / `Rally Order`
  - Operator Major Operation `drone_strike` / `Drone Strike`
- `rally_order`:
  - `marshal_alpha` only
  - cost 1 Command
  - usable during allied stack activation before battle completion
