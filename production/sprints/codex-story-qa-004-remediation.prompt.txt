Research-backed UI direction:
Use Unity's built-in uGUI layout system rather than hand-positioned map TextMesh/buttons.
- Unity uGUI Auto Layout provides Horizontal/Vertical/Grid layout groups and layout elements to place children in structured layouts with padding/spacing and content sizing.
- VerticalLayoutGroup stacks child elements with spacing/padding and sizes from min/preferred/flexible dimensions.
- GridLayoutGroup places child elements into fixed cells and supports spacing/constraints.
This is the appropriate out-of-the-box solution for prototype command panels and button groups.
